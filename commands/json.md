name: json
description: Format, validate, or generate JSON schema
---

Work with JSON data:

**Format/Prettify:**
- Indent with 2 or 4 spaces
- Sort keys alphabetically
- Compact (minify)

**Validate:**
- Check for syntax errors
- Validate against a schema

**Generate Schema:**
Create JSON Schema from example data:
- Infer types
- Detect required fields
- Add descriptions

**Transform:**
- JSON ↔ YAML
- JSON ↔ TOML
- JSON ↔ CSV (for arrays)
- Flatten/Unflatten nested objects

**Query/Filter:**
- JSONPath expressions
- jq-style queries
- Filter arrays

Provide the JSON and what you want to do with it.
