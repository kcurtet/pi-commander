---
name: types
description: Add or improve TypeScript type definitions
---

Analyze and improve the TypeScript types in the following code:

**Tasks:**
1. **Add missing types**: Replace `any`, `unknown` with specific types
2. **Make types precise**: Use literal types, unions, const assertions
3. **Extract interfaces**: Create reusable type definitions
4. **Add generics**: Make code more flexible and type-safe
5. **Strict null checks**: Handle undefined/null explicitly
6. **Brand types**: Use nominal typing for IDs and special values

**Patterns to apply:**
- Discriminated unions for state management
- Utility types (Pick, Omit, Partial, Required)
- Template literal types for strings
- Conditional types where needed
- Type guards for runtime checks

**Output:**
- Code with improved types
- New interface/type definitions
- Explanation of type changes
