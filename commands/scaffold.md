Generate boilerplate code for the requested pattern:

**Available scaffolds:**

**Frontend:**
- React component (functional, with hooks)
- Vue component (composition API)
- Svelte component
- Next.js page/api route

**Backend:**
- Express route/middleware
- FastAPI endpoint
- Go HTTP handler
- Rust actix-web handler

**Tests:**
- Unit test file
- Integration test
- E2E test (Playwright/Cypress)

**Configuration:**
- Dockerfile
- docker-compose.yml
- GitHub Actions workflow
- ESLint/Prettier config

**Specify:**
1. What you want to scaffold
2. Language/framework
3. Any specific requirements

Output will be production-ready boilerplate with:
- Proper structure
- Type definitions
- Error handling
- Comments
- Example usage
