Help with SQL queries:

**Explain:** Describe what the query does in plain language

**Optimize:** Suggest improvements for performance
- Index recommendations
- Query restructuring
- JOIN optimization
- Subquery → CTE → JOIN conversions

**Translate:** Convert between SQL dialects
- PostgreSQL ↔ MySQL
- SQLite ↔ PostgreSQL
- Any SQL ↔ ORM (Prisma, Sequelize, SQLAlchemy, etc.)

**Debug:** Find issues in queries
- Syntax errors
- Logic errors
- Performance bottlenecks

**For each query analyzed:**
1. Plain English explanation
2. What tables/columns are involved
3. Estimated complexity
4. Optimization suggestions (if applicable)
5. Index recommendations (if applicable)

Include the dialect when asking (PostgreSQL, MySQL, SQLite, etc.)
