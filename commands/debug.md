name: debug
thinking: high
description: Analyze errors and tracebacks to find root cause and solutions
---

Analyze the following error or traceback and help debug it:

1. **Identify the root cause**: What is actually failing and why
2. **Explain the error**: In simple terms, what does this error mean
3. **Find the location**: Point to the exact file and line causing the issue
4. **Propose solutions**: List possible fixes with code examples
5. **Prevent recurrence**: Suggest how to avoid this error in the future

If you need more context, ask for:
- The code around the error location
- Input data that triggered the error
- Recent changes to the codebase

Be systematic: start from the error message, trace back through the stack, and identify the failing assumption.
