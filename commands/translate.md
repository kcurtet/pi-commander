**Target language:** (specify or I'll detect from context)

**Translation guidelines:**

1. **Idiomatic code**: Use language-specific idioms and patterns
   - Not just syntax translation
   - Use standard libraries where appropriate
   - Follow language conventions

2. **Type systems**: Adapt typing appropriately
   - TypeScript → Python: Use type hints
   - Python → Rust: Leverage ownership system
   - Any → Go: Use Go idioms (error handling, interfaces)

3. **Ecosystem**: Use appropriate libraries
   - HTTP: fetch/axios/requests/reqwest
   - JSON: native/serde/simplejson
   - Testing: jest/pytest/cargo test

4. **Output:**
   - Translated code
   - Notes on any significant changes
   - Required dependencies/packages

Common translations:
- TypeScript ↔ Python
- JavaScript ↔ Python
- Python ↔ Go
- Any language → Rust
- Java ↔ Kotlin
